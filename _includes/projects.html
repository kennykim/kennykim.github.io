Qar
  • Java
  • Artificial Intelligence
  • Q-Learning

Qar is a small project AI-based system in which a car in a grid learns how to drive avoiding walls. It uses Q-Learning algorithm to obtain the optimal policy via trial and error. The car has the option to choose one action before every tick and the car will move one square into the facing direction after the tick.

RPi Motion detection security camera
  • Python
  • Computer Vision
  • RPi
  • OpenCV

Simple security project using a Raspberry Pi with a camera. The code detects motion and sends an email with the video feed to the user.

Motion is detected by using the difference between two consecutive frames, and if the difference is substantial enough, then the camera stard saving those frames into a video. Once there is no movement, the video is closed and sent to the user.

Msp430 Racing Car Game
  • Msp430
  • C
  • Interrupts
  • Embedded Systems

Simple racing car game created for the Msp430 with an attached LCD black and White display. The goal is to void as many cars as possible using the right and left buttons to move right and left. The graphics were created using pixel art.

The game is developed in C in its entirety from handling pins to writing pixels to the screen to the abstracted game logic of detecting overlaps and accelerating the speed of the game.
