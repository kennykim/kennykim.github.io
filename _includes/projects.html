Qar
  • Java
  • Artificial Intelligence
  • Q-Learning

Qar is a small project AI-based system in which a car in a grid learns how to drive avoiding walls. It uses Q-Learning algorithm to obtain the optimal policy via trial and error. The car has the option to choose one action before every tick and the car will move one square into the facing direction after the tick.

Action Space

  • Turn Left
  • Turn Right
  • Do Nothing

State Space

  • Distance from wall in front
  • Distance from wall to the left
  • Distance from wall to the right

Rewards

  • -100 for crashing
  • 0 for staying alive

Yelp review star predictor
  • Python
  • Data Mining
  • Document classification

This project was developed for the Yelp dataset challenge, it is a star predictor that uses Naive Baye's classification algorithm to classify a review into how many stars it has. The data was taken from Yelp webpage and half is used to train the classificator and the other half is used to test the classificator. The results of the script are in number of correct and incorrect results and the error distance, which is a metric of how far the actual classification was from the predicted one.

Results

The classificator is able to predict correctly about 40% of the reviews, with an error distance of 2
